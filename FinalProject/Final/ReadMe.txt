Click to "Get ToDo" for all current info. Click "back" to go back to main page.

"Post ToDo" input box is for entry info and then click save.

Put modified info, click "modify". Type the id into the box and a new title in the other box for modified info. Click "modify" or "back".

For delete, you the following method as listed below (partially posted on web, too):
http://localhost:8080/{id}
This will delete an id if it exists.

Example
http://localhost:8080/3
Then enter. This would delete the info with the ID of 3.
